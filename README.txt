Funcionalidades da Calculadora
Adicionar Alimentos:

Inserir nome do alimento

Inserir valores nutricionais (calorias, proteínas, carboidratos e gorduras)

Validação para garantir que o nome do alimento seja fornecido

Lista de Alimentos:

Exibe todos os alimentos adicionados em uma tabela

Mostra os valores nutricionais de cada alimento

Permite remover alimentos individuais

Cálculo de Totais:

Soma automática de todas as categorias nutricionais

Exibe os totais de forma clara e organizada

Armazenamento Local:

Os dados são salvos no localStorage do navegador

Persistência dos dados mesmo após atualizar a página

Design Responsivo:

Layout adaptável para diferentes tamanhos de tela

Estilo limpo e moderno

Como Usar
Preencha os campos com as informações do alimento

Clique em "Adicionar Alimento" para incluir na lista

Repita o processo para todos os alimentos que deseja calcular

Veja os totais nutricionais atualizados automaticamente

Use o botão "Remover" para excluir alimentos específicos

Use "Limpar Tudo" para reiniciar a calculadora

Esta calculadora é perfeita para quem quer acompanhar a ingestão nutricional diária ou planejar refeições com valores nutricionais específicos.
